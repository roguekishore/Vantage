## Vantage LeetCode Sync Extension

Sync your LeetCode progress directly with Vantage.

### What this extension does
- Imports your **existing solved LeetCode problems** into Vantage.
- Automatically syncs **newly solved problems** going forward.
- Keeps your Vantage progress up to date with minimal setup.

### Installation (Chrome)
1. Download the ZIP from this release.
2. Extract the ZIP to a folder.
3. Open `chrome://extensions/`.
4. Turn on **Developer mode** (top-right).
5. Click **Load unpacked**.
6. Select the extracted extension folder.

### Notes
- This is a **manual install** build (not from Chrome Web Store).
- After loading, open LeetCode and Vantage in your browser to start syncing.
